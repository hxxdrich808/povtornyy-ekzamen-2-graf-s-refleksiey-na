# Project Title

This project demonstrates the use of the `rich` library for enhanced console output in Python.

## Requirements

- Python 3.8+
- `rich` library (specified in `requirements.txt`)

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the main script:

```bash
python src/main.py
```

You should see a colored output message in the console.
